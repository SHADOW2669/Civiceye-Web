Civic Eye

Introduction

CivicEye is an advanced AI-powered software designed to detect and monitor traffic violations using CCTV cameras. It enables users to identify infractions such as riding without a helmet or triple riding in real-time. The system seamlessly integrates with existing CCTV setups, processes violations locally for enhanced privacy, and securely transmits critical data to a centralized database. Users can access detailed violation logs, reports, and analytics through an interactive dashboard, making traffic monitoring more efficient and data-driven.

MORE INFO

OBJECTIVES

AI-Powered Traffic Violation Detection: Utilize YOLOv8 to detect violations such as "No Helmet," "Triple Riding," and other infractions in real time.
Automated Data Reporting: Capture and transmit violation details (images, timestamps, and types) from the client’s system to a centralized database.
Centralized Monitoring & Management: Enable users to track, review, and manage detected violations through a web-based dashboard.
Privacy-Focused & Efficient Processing: Perform detection locally on the client’s machine, ensuring data privacy while minimizing server costs.
Scalable & User-Friendly Deployment: Develop a robust, easy-to-install software that runs across multiple devices with minimal setup.

KEY FEATURES

Real-Time Violation Detection: Automatically detect and record traffic violations using the YOLOv8 object detection model.
Local Processing: Run the detection software locally on the client’s machine, minimizing data transmission.
Automated Reporting: Send violation evidence (photos and metadata) to a centralized database for storage and review.
Web Dashboard: Provide a user-friendly website for viewing, searching, and filtering violations.
Customizable Configuration: Users can easily configure the CCTV feed (RTSP URL) and software settings.

SYSTEM REQUIREMENTS

Processor: Intel i3 or higher
RAM: 4GB (8GB recommended)
GPU: Optional (YOLOv8-tiny can run on CPU)
Storage: 20GB free disk space
